# Project - Quzee
Description : An web application to create quiz forms, self quiz, etc.

### Features :
-	Login with google
-	User can create Form and publish and see the response and give scores
-	Create questions with AI
-	create self quiz with AI 

### Type of forms : 
-	Feedback collectors: no scoring, anonymous feedback also accepted
-	Quiz : validate scores
-	Schecduled Forms

### Add ons : 
-	Question Data types  : 
o	short answer, paragraph, check boxes, radio, true or false, rating
-	Options : 
o	timer, expire date, question required, save responses, shuffle questions, require login (anonymous), allow retakes

### Technologies using : 
-	REACT  for web sied
-	Firebase for storage & auth
-	Claude API for AI generating questions
-	Github for deployment

### Starting with create react app
- check for node > main terminal > node --version (ref : https://youtu.be/lt5D2EWZMN0?si=3-2XcWE-WgRmLQBm)
- open folder in VS code > terminal > npx create-react-app quzee
- To open react app > npm run start

### Installing firebase 
- Go to firebase console and create app
- Go to VS code > terminal > npm install firebase
- create file > firebase.js under src folder and paste the Firebase SDK
